Cara edit:
1. Buka config.js
2. Ganti letterMessage untuk isi surat
3. Ganti photos dengan link foto sendiri atau file lokal assets/foto1.jpg
4. Isi music jika punya file mp3, contoh: music: "assets/music.mp3"
5. Buka index.html di browser

File utama:
- index.html
- style.css
- script.js
- config.js
